## tailwind css

1. 라이브러리 설치
   | npm install -D tailwindcss postcss autoprefixer

2. 태일윈드 초기화
   | npx tailwindcss init -p

3. 태일윈드 설정
   tailwind.config.js 파일

4. 글로벌 css 에 태일윈드 적용
